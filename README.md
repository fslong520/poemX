# poemX
诗界

## 项目简介
poemX（诗界）是一个将中小学必背古诗文转换为视频内容的项目。通过现代化的视听技术，让古典诗词焕发新的生命力，帮助学生更好地理解和记忆传统文化。

## 项目结构
```
中小学必背古诗文/
├── 01-一年级/          # 一年级古诗文
├── 02-二年级/          # 二年级古诗文
├── 03-三年级/          # 三年级古诗文
├── 04-四年级/          # 四年级古诗文
├── 05-五年级/          # 五年级古诗文
├── 06-六年级/          # 六年级古诗文
├── 07-七年级/          # 七年级古诗文
├── 08-八年级/          # 八年级古诗文
├── 09-九年级/          # 九年级古诗文
├── 10-高中/            # 高中古诗文
└── 补充篇目/           # 补充古诗文
```

## 内容特色
- **完整覆盖**：包含教育部2024最新版中小学必背古诗文345篇
- **结构清晰**：按年级分类，便于查找和学习
- **内容丰富**：每首诗包含原文、译文、作者信息和详细赏析
- **格式统一**：标准化的Markdown格式，便于后续处理

## 技术特点
- 基于现代化视频制作技术
- 支持批量处理和自动化生成
- 保留古典韵味的同时融入现代元素
- 适合教学和自主学习使用

## 使用说明
1. 浏览对应年级文件夹查找所需古诗文
2. 每首诗的md文件包含完整的内容信息
3. 可根据需要提取内容用于视频制作
4. 支持按诗人、篇目等多种方式检索
